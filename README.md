# For-NCU-MIS

一鍵產出付費軟體使用時數統計報表

---
# 準備工作
將 labs.csv, students.csv, usage.csv 三個檔案放在同一個資料夾下。  
開啟一個新的 Excel 活頁簿，按下 Alt + F11 進入 VBA 編輯器。  
點選選單 [工具] (Tools) -> [引用項目] (References)。  
勾選 Microsoft ActiveX Data Objects 6.1 Library (若沒有 6.1，選 2.8 或最新版本即可)。  

# 測試檔案（test_dataset）介紹
1. labs：實驗室經費代碼
2. students：學生資料
3. usage：軟體使用紀錄
註：學校系統下載之檔案與測試檔案類似
